# RVM使用总结

### 是什么？
> RVM is a command-line tool which allows you to easily install, manage, and work with multiple ruby environments from interpreters to sets of gems.

总结：
`命令行工具` `ruby版本管理` `gem`

### 怎么用？
#### step1：install
[官网](https://rvm.io/)在这里，这里就不赘述了

#### step2：常用命令

##### - 安装依赖库
`rvm requirements`
##### - 安装制定版本Ruby
`rvm install x.x`
##### 切换ruby版本
`rvm use x.x [--default]`
